StylesHub Pro — Salon Management System

Built by SME Systems Studio | Patience Oduori

A cloud-based salon management system with a public client self-booking portal. Staff manage the business from the main app while clients book appointments themselves via a separate public link.

Staff login: admin / admin123 or cashier / cashier123
Booking portal: no login required

✨ Features

Point of Sale with service images and categories
Appointment booking with double booking prevention
Client self-booking portal — clients book online via WhatsApp link
Stylist management and performance tracking
Client profiles with visit history
Sales history with digital receipts
Expenses tracking
Profit & Loss and stylist performance reports
Secure login with role-based access

🛡️ Security

SHA-256 password hashing
LockService concurrency protection
Audit trail on every transaction
Client portal has no access to staff data

🔧 Tech Stack
Google Apps Script · Google Sheets · HTML · CSS · JavaScript · Chart.js
